# 📊 Projects

## 1️⃣ Global Market Regime Detection

Detects the **current global financial market regime** using a Hidden Markov Model trained on multi-market financial data.

Markets analyzed include:

- S&P 500
- Euro Stoxx 50
- Nifty 50
- Nikkei 225
- VIX (volatility index)

Key features:

- Global return indicators
- Cross-market dispersion
- Volatility estimation
- Trend indicators
- Regime transition modeling

Location:

```
market-regime-detection/
```
